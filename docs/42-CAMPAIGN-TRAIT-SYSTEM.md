**GRAFFITI KINGS – CAMPAIGN TRAIT SYSTEM (EXPANDED)**

**Important Clarification:**
This system is **NOT** about generating 30-day tasks for users to create PFPs or NFTs. It is about using a modular trait system to create thousands of unique **campaign advice, strategies, and execution plans** from the brain’s knowledge base.

**Core Philosophy:**
Even if the app only had 30 days of base content, the Trait System allows it to generate 100,000s or even millions of unique variations — making every user’s experience feel personalized and infinite.

---

### **Trait Categories (PFP-Style Layers)**

**Core Traits (Always Present):**
- Location Type (alley, bridge, rooftop, pavement, abandoned building, high street, train station, park, underpass, etc.)
- Time of Day (dawn, morning, golden hour, dusk, night, late night)
- Technique (wheatpaste, stencil, reverse graffiti, chalk, sticker, projection, yarn bombing, mixed media)

**Modifier Traits (Mix & Match):**
- Visual Style (minimal, chaotic, cinematic, raw, clean, psychedelic, vintage, futuristic)
- Tone / Message (angry, poetic, sarcastic, hopeful, chaotic, mysterious, defiant, humorous, emotional)
- Risk Level (very low, low, medium, high, very high)
- Audience Target (local community, youth, older generation, activists, online only, brands, general public)
- Monetization Angle (free drop, paid prints, Patreon, brand collabs, service offering, NFT drop, donation drive)
- Documentation Style (professional photos, raw phone footage, time-lapse, before/after, story format, mini doc)
- Viral Hook (surprise, emotion, controversy, mystery, relatability, process, reaction)

**Advanced / Rare Traits:**
- Lore Integration (HODL WARRIORS faction, personal mythology, ongoing story)
- Collaboration Style (solo, small crew, community call-out, brand partnership)
- Tech Integration (QR codes, AR filter, simple web app, token gating)
- Seasonal / Timely (election season, holiday, local event, global trend)

**Rarity System:**
- Common traits (high chance, safe combinations)
- Rare traits (medium chance, stronger impact)
- Ultra-rare traits (low chance, high reward, special combinations)

---

### **How the System Generates Unique Plans**

1. Start with one of the 5 Base Templates
2. Apply 4–8 random but intelligent trait combinations
3. Adjust based on user context (location, experience, goals, risk tolerance)
4. Output a complete 30-day plan that feels custom-made

**Example Output:**
User in Manchester, beginner, wants to build audience + low risk
→ Base Template: Raw Drop
→ Traits: Location (high street at dusk) + Technique (wheatpaste + chalk) + Tone (hopeful + defiant) + Viral Hook (before/after transformation) + Documentation (raw phone footage + story format)

Result: A completely unique 30-day plan that no other user will get exactly the same way.
