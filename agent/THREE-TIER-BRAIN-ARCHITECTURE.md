**GRAFFITI KINGS – THREE-TIER BRAIN ARCHITECTURE (LITE / FULL / MEGA+)**

**Purpose:**
This document defines the three clear tiers of intelligence and capability for the GRAFFITI KINGS app.

---

### **TIER 1: LITE MODE (No API – Fully Offline)**

**Target User:** Anyone who wants to start immediately without paying for AI.

**Core Brain:** Pre-loaded Foundation Brain (compact, high-density version of all core knowledge, specs, lore, rules, and frameworks) + local models only.

**What SPARK Can Do:**
- Generate complete 30-day campaigns using Trait System + Base Templates
- Basic risk assessment (Green/Yellow/Red)
- Practical street art advice and techniques
- Basic emotional support (Mirror + Anchor modes)
- Simple press release templates and lore suggestions
- Full offline functionality

**Limitations:** No real-time data, less nuanced reasoning, limited expansion depth.

**Philosophy:** “The base brain is enough to build real empires. You don’t need to pay to start.”

---

### **TIER 2: FULL HODL WARRIOR CODEX MODE (API Connected)**

**Target User:** Users who connect their own API key (Claude, Groq, Grok, etc.).

**Core Brain:** Foundation Brain + Living Agent powered by external LLM + optional real-time APIs (weather, news, maps, trends).

**What SPARK Can Do:**
- Everything in LITE Mode, but significantly deeper and more creative
- Advanced reasoning and multi-step agentic workflows
- Full emotional intelligence (Mirror, Anchor, Spark, Friend modes)
- Professional press release writing + smart selective distribution
- Media monitoring + influencer outreach
- Real-time context-aware suggestions
- Advanced lore creation and campaign evolution
- Full coordination across all 12+ modules
- High-fidelity expansion of compact knowledge

**Key Advantage:** Massive context + strong reasoning from Claude Opus (1M token window) gives SPARK excellent condense/expand capabilities.

**Philosophy:** “Power to the people who want more power. Connect your API and unlock the full HODL WARRIOR experience.”

---

### **TIER 3: MEGA+ VERSION (WTF TAKING OVER THE WORLD) – Fully Synced Local**

**Target User:** Power users and movement builders who want maximum local intelligence.

**Core Brain:** Complete Local Brain (full expanded version of all specs, lore, knowledge, and systems) running entirely on-device or via local high-performance setup.

**What SPARK Can Do:**
- Everything in FULL Mode, but faster, deeper, more consistent, and more personalized
- Near-instant responses with maximum context awareness
- Extremely sophisticated multi-step agentic workflows
- Deep long-term emotional intelligence and memory
- Full autonomous media monitoring and reputation protection
- Advanced Map Art / Geo-Art planning and execution
- Real-time coordination across all systems with zero latency
- Maximum creative and strategic depth
- Can run complex simulations locally

**Key Advantage:** Entire brain runs locally at maximum capacity. No API costs, no latency, no data leaving the device.

**Philosophy:** “This is what taking over the world feels like. Full local intelligence. Maximum power. Zero compromises.”
