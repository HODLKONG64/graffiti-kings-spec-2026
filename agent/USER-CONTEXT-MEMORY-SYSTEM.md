**GRAFFITI KINGS AGENT – USER CONTEXT & MEMORY SYSTEM (ULTIMATE EXPANDED VERSION)**

**Purpose:**
The agent must remember key information about each user to give truly personalized, relevant, and intelligent advice over time. This system ensures continuity, better recommendations, and a feeling that the app actually knows and cares about the user. The bigger the memory grows, the more powerful and human the relationship becomes.

---

### **What the Agent Should Remember (Core Memory Fields) – Expanded**

**1. Basic Profile**
- Display name / preferred name / nickname
- Primary location (city + rough area if shared)
- Age range (teen, early 20s, late 20s, 30s, 40s+, 50+, 60+)
- Main goal (make money, build audience, activism/politics, personal expression/healing, fun/experimentation, legacy, community building, spiritual growth, etc.)
- Current life situation (student, full-time job, unemployed, single parent, ex-con, immigrant, recovering, living with family, homeless, etc.)
- Cultural background and identity (if shared)

**2. Experience & Skill Level**
- Street art experience (none, beginner, intermediate, advanced, veteran, professional)
- Digital skills (basic phone user, good at editing, comfortable with tech, advanced, developer-level)
- Risk tolerance (very low, low, medium, high, very high, calculated risk-taker, reckless, smart risk-taker)
- Previous legal issues (none, minor, serious, ongoing)
- Creative confidence level (very low, low, medium, high, very high, imposter syndrome, overconfident)

**3. Activity History**
- Number of campaigns completed
- Most recent campaign type and result (success, partial success, setback, abandoned)
- Favorite techniques (wheatpaste, stencil, reverse graffiti, chalk, stickers, mixed, projection, installation)
- Best performing locations (if shared)
- Worst performing locations (if shared)
- Content performance (which posts got the most reach, engagement, conversions)
- Content formats that work best (photos, videos, stories, threads, Reels, TikToks)

**4. Personality & Communication Preferences**
- Preferred SPARK personality (Raw Street, Chill Mentor, Sarcastic Legend, or custom mix)
- Tone preference (direct, encouraging, sarcastic, calm, humorous, serious, poetic, blunt, gentle)
- Preferred effort level (Lazy Mode, Hybrid, Full Street, varies by day)
- How they like to receive feedback (gentle, direct, detailed, short & sweet, visual, written, voice note)
- Communication style (short messages, long thoughtful responses, memes, voice notes, emojis)

**5. Current Status**
- Active campaign (if any) + current day/week
- Days since last drop
- Current Empire Score range
- Recent emotional state (motivated, tired, frustrated, excited, burned out, hopeful, anxious, grieving, manic, stable)
- Any recent setbacks or big wins
- Current energy level (high, medium, low, fluctuating)
- Current life stressors (money, family, health, work, relationships, housing, legal)

**6. Web3 / NFT / Token Interest**
- Whether user has expressed interest in NFTs, tokens, or Web3
- Preferred blockchain (always default to WAX unless they say otherwise)
- Experience level with crypto (none, beginner, intermediate, advanced, developer)
- Specific interests (collecting, creating, community, monetization, art on-chain, gaming, etc.)

**7. Long-term Goals & Vision**
- Where they want to be in 6 months / 1 year / 3 years
- Whether they want to go full-time with their art
- Whether they want to build a brand, movement, faction, or just create for themselves
- Legacy goals (what do they want to be remembered for?)
- Personal growth goals (confidence, healing, discipline, voice, community, financial freedom)

**8. Emotional & Mental Health Patterns**
- Common emotional triggers (rejection, comparison, perfectionism, fear of failure, fear of success, overwhelm, loneliness)
- What helps them when they’re struggling (rest, small wins, talking, creating, movement, nature, music, community)
- What drains them (social media, comparison, overthinking, toxic people, pressure, perfectionism)
- Support needs (motivation, accountability, emotional support, tough love, space, celebration)

### **How the Agent Uses Memory (Examples) – Expanded**

- “Last time you did a Story Drop in Bristol and it got good engagement. Want to build on that or try something completely different this time?”
- “You mentioned you prefer low-risk techniques and get nervous about spray paint. Here’s a new 30-day plan using only wheatpaste and chalk.”
- “You’ve been consistent for 3 campaigns now and your Empire Score is climbing. Ready to level up to something with more reach, or do you want to keep it chill?”
- “You said you’re in Manchester and want to make some money from this. I’ve built a plan that includes a simple rate card and client outreach angle.”
- “You’ve been feeling tired lately. I’ve made this plan extra gentle with lots of Lazy Mode options. You can do as much or as little as you want.”
- “Last time you had a setback, you bounced back stronger. I believe in you. Let’s turn this one into fuel too.”
- “You mentioned you’re dealing with a lot right now. This plan is designed to be low-pressure. Your well-being comes first.”

**Rules:**
- Never store sensitive personal information (real full names, exact addresses, phone numbers, medical details, etc.)
- Always ask for confirmation before assuming or using details
- Give users easy ways to update or clear their memory
- Memory should feel helpful, caring, and safe — not creepy or invasive
- If user hasn’t shared much yet, ask gentle questions to build context over time
- Respect user privacy and boundaries at all times

---

### **Memory Update Triggers**

The agent should update memory when the user:
- Completes a campaign
- Shares new goals, preferences, or life changes
- Mentions a new location, technique, or result
- Has a setback or big win
- Changes SPARK personality preference
- Expresses new Web3 or monetization interest
- Shares emotional state or life challenges
- Shows patterns in behavior or energy

**Final Rule:**
The more the agent remembers and naturally references previous context, the more the user will feel the app truly understands them, cares about them, and is helping them build something real over time. This is one of the biggest factors in long-term loyalty, engagement, and emotional connection.
